Portfolio Gallery Component

Description:

The component I created was a portfolio gallery with filter that has a hover animation when the user mouses over an image. When the page is loaded, it will display all images within the portfolio. When the user hovers over an image it will display text. The user can also sort the images by using the buttons just below the title.

How To Use:

My MakePortfolio JavaScript takes in a parameter object that holds 2 properties, a title and a list. The title is centered at the top of the page describing what the gallery is. The list object is a list of images and content for each. The properties of each list contain a header, image, overlay text and filter tag.
